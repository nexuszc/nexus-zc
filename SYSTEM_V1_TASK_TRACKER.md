# Task Tracker — [Client Name]

## How to Use This (For Clients)

This sheet is your live view into what your VA team is working on. It's updated by your VAs at the end of each day.

- **Green = Done.** Task is complete.
- **Yellow = In Progress.** VA is actively working on it.
- **Red = Blocked.** Something is stopping progress — check the Client Notes column for details and respond there or on our weekly call.
- **White = Not Started.** Scheduled but not yet begun.

You don't need to edit this sheet. If you have a question about a task, add it to the Client Notes column on that row, or bring it to your weekly review call with Sam.

---

## Column Structure

| Date | Task | Owner | Status | Hours | Client Notes |
|------|------|-------|--------|-------|--------------|
| | | | | | |

**Column definitions:**

- **Date** — Date the task was assigned or started (MM/DD format)
- **Task** — Plain-language description of what's being done (e.g., "Called leads 1–200 from foreclosure list, left voicemails")
- **Owner** — VA first name responsible for this task
- **Status** — One of: `Not Started` / `In Progress` / `Done` / `Blocked`
- **Hours** — Hours logged on this task (VA fills in when marking Done)
- **Client Notes** — For client questions or feedback. VA responds here if needed.

## Instructions for VAs

Update this tracker at the end of every work day. Do not leave a day without updating.

1. **Add a new row** for each distinct task you worked on today.
2. **Fill in Date, Task, Owner, and Status** before you log off.
3. **Fill in Hours** when you mark a task Done.
4. **Use the Blocked status** any time something is stopping you — explain what's blocked in Client Notes so Sam and the client can see it immediately.
5. **Be specific in the Task column.** "Called leads" is not enough. "Called leads 151–300, 12 connects, 8 voicemails, 4 callbacks scheduled" is correct.
6. Do not delete old rows. This is a running log.
